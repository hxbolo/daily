## 项目结构
 ### project
- dist    // 打包编译后的文件
- product // 公共（模版）文件
- src             // 项目文件    
- api       // 封装的axios请求
- assets    // 静态资源
- components    // 自定义组件
- config    // 项目配置文件，环境和appid配置
- icons    // svg 和iconfont
- layout    // 废弃
- mixins    // 混入
- plugins    // 插件
- router    // 路由
- store    // vuex
- style    // 公共样式
- ui    // 基础封装，tabbar、icon、scroll
- utils    // 工具类
- view    // 页面
- App.vue    // 根页面
- main.js    // 入口文件
- permission.js    // 路由拦截
- vue.config.js    // webpack配置
